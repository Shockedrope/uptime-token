## Overview

Uptime ($UPTI) is a fully deployed ERC-20 token on the Polygon network. It provides a simple way to trade time and services with friends and family. Set your own exchange rate, send tokens freely, and settle up on your terms.

## How It Works

1. **Add to MetaMask** — Import the $UPTI token using the contract address above. Make sure you're on the Polygon network.
2. **Set your rate** — Decide what UPTI is worth to your group. 100 UPTI per hour? 50 per errand? You set the rules.
3. **Send and receive** — Transfer UPTI directly to anyone's wallet. No middlemen, no fees worth mentioning, near-instant settlement.
4. **Redeem services** — Spend your UPTI on whatever you've agreed on. Lawn mowing, babysitting, tech support — the economy is yours.

## Features

- ⛏ **Mintable** — The owner can create new UPTI tokens anytime, up to the 1,000,000 cap.
- 🔥 **Burnable** — Any holder can permanently destroy their tokens, reducing total supply.
- 🔒 **Supply cap** — Hard limit of 1,000,000 UPTI — no runaway inflation, ever.
- ⚡ **Near-zero fees** — Polygon transactions cost fractions of a penny. Send freely.
- 🌐 **Permissionless** — Anyone with a wallet can hold and transfer UPTI. No signups, no gatekeepers.
- 🔍 **Transparent** — Every transaction is public and auditable on Polygonscan forever.
